<h2>1. Introduzione al C++</h2>
            <p>Il C++ è un linguaggio di programmazione ad alto livello, estremamente potente e utilizzato per creare sistemi operativi, videogiochi e software professionali.</p>
            
            <h3>Struttura base di un programma</h3>
            <p>Ogni programma C++ segue uno schema standard. Vediamo il classico "Hello World":</p>
            
            <pre><code>
#include &lt;iostream&gt; // Libreria per input/output

using namespace std; // Evita di scrivere std:: ogni volta

int main() {
    // Il corpo del programma inizia qui
    cout << "Ciao Mondo!" << endl; // Stampa a video
    
    return 0; // Indica che il programma è terminato correttamente
}
            </code></pre>
            <p><strong>Analisi riga per riga:</strong></p>
            <ul>
                <li><code>#include &lt;iostream&gt;</code>: Carica le funzioni di base come <code>cin</code> e <code>cout</code>.</li>
                <li><code>int main()</code>: È il "cuore" del programma; l'esecuzione parte da qui.</li>
                <li><code>cout</code>: L'oggetto che si occupa di inviare dati allo schermo.</li>
            </ul>
